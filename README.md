# Research Workspace

本仓库已按 `代码 / 文档 / 产物` 分层整理。

## 目录结构

- `scripts/`
  - 抓取与分析脚本
- `docs/`
  - 使用说明与项目文档
- `requirements/`
  - 依赖清单
- `outputs/`
  - 已生成的论文抓取结果与图表

## 快速开始

安装依赖:

```bash
pip install -r requirements/requirements_icml_neurips_iclr_2025.txt
```

运行主脚本:

```bash
python3 scripts/icml_neurips_iclr_2025_keyword_report.py
```

详细用法见:

- `docs/ICML_NeurIPS_ICLR_2025_keywords_README.md`
